# Multiple Proto & Service

#### Struktur Folder:
- **/proto**
  - `order.proto`
  - `dll.proto`
- **/services**
  - `order.service.js`
  - `dll.service.js`
- `client.js`
- `protos.js`
- `server.js`
- `test.js`


#### Keterangan:

> `client.js` Managemen untuk koneksi ke gRPC server.

> `protos.js` Managemen untuk Protobuf gRPC.

> `server.js` Main project gRPC Server.

> `test.js` contoh consume gRPC Server.
